## 🎨 자주 쓰는 값들

### 팀 카테고리 (team_category)
```
'전도팀'      - 전도 담당
'찬양팀'      - 찬양 담당
'중보기도'    - 기도 담당
'푸드'        - 음식 담당
'차량물품'    - 차량과 물품 관리
'부채춤'      - 부채춤 공연
'미용'        - 미용 담당
'목사'        - 목사님
'장로'        - 장로님
'총괄팀장'    - 총괄 책임자
```

### 역할 (role)
```
'leader'      - 팀장, 리더
'member'      - 팀원
'volunteer'   - 자원봉사자
```

### 일정 타입 (event_type)
```
'press'       - 기자회견
'rally'       - 집회
'concert'     - 음악회, 콘서트
'outreach'    - 전도, 아웃리치
'meeting'     - 회의, 모임
'other'       - 기타
```

### 사역 카테고리 (category)
```
'preparation' - 준비 작업
'event'       - 행사 진행
'followup'    - 사후 관리
'logistics'   - 물류, 장비
'program'     - 프로그램
```

### 우선순위 (priority)
```
'low'         - 낮음 (여유있게)
'medium'      - 보통 (적당히)
'high'        - 높음 (중요해요!)
'urgent'      - 긴급 (지금 당장!)
```

### 상태 (status)
```
'todo'        - 할 일 (아직 안 함)
'in_progress' - 진행 중 (하는 중)
'completed'   - 완료 (다 했어요!)
'cancelled'   - 취소 (안 해요)
```

## 📝 데이터 확인하기

### 전체 데이터 보기
```sql
-- 참가자 전체
SELECT * FROM participants WHERE is_active = true ORDER BY created_at DESC;

-- 일정 전체
SELECT * FROM schedules WHERE is_public = true ORDER BY start_time;

-- 사역 전체
SELECT * FROM tasks WHERE is_public = true ORDER BY due_date;
```

### 데이터 개수 세기
```sql
SELECT 
  '참가자' as 항목, COUNT(*) as 개수 FROM participants WHERE is_active = true
UNION ALL
SELECT '일정', COUNT(*) FROM schedules WHERE is_public = true
UNION ALL
SELECT '사역', COUNT(*) FROM tasks WHERE is_public = true;
```

### 최근 추가된 데이터 보기
```sql
-- 최근 추가된 참가자 5명
SELECT name, team_category, created_at 
FROM participants 
WHERE is_active = true 
ORDER BY created_at DESC 
LIMIT 5;

-- 곧 있을 일정 5개
SELECT title, start_time, location
FROM schedules
WHERE is_public = true AND start_time > NOW()
ORDER BY start_time
LIMIT 5;

-- 급한 사역 5개
SELECT title, priority, due_date
FROM tasks
WHERE is_public = true AND status != 'completed'
ORDER BY 
  CASE priority
    WHEN 'urgent' THEN 1
    WHEN 'high' THEN 2
    WHEN 'medium' THEN 3
    WHEN 'low' THEN 4
  END,
  due_date
LIMIT 5;
```

## 🗑️ 데이터 수정/삭제하기

### 데이터 수정
```sql
-- 참가자 정보 수정
UPDATE participants
SET 
  phone = '010-9999-9999',
  team_category = '찬양팀',
  position = '찬양팀원'
WHERE name = '홍길동';

-- 일정 시간 변경
UPDATE schedules
SET 
  start_time = '2025-12-25 20:00:00+09',
  end_time = '2025-12-25 22:00:00+09'
WHERE title = '크리스마스 축제';

-- 사역 상태 변경
UPDATE tasks
SET status = 'completed'
WHERE title = '무대 설치';
```

### 데이터 삭제
```sql
-- 참가자 비활성화 (삭제하지 않고 숨기기)
UPDATE participants
SET is_active = false
WHERE name = '홍길동';

-- 일정 완전 삭제
DELETE FROM schedules
WHERE title = '취소된 일정';

-- 사역 완전 삭제
DELETE FROM tasks
WHERE title = '취소된 사역';
```

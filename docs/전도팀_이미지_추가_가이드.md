# 전도팀 이미지 추가 가이드

## 📸 작업 내용

전도팀 사역 사진 6장을 전도팀 갤러리에 추가합니다.

## 📋 이미지 저장하기

### 1️⃣ 이미지 파일 저장

첨부하신 6장의 사진을 다음과 같이 저장해주세요:

**저장 위치:** `public/images/` 폴더

| 첨부 이미지 | 저장할 파일명 | 설명 |
|----------|--------------|------|
| 1번째 사진 | `evangelism-team-1.jpg` | 줌 미팅 - 전도팀 회의 |
| 2번째 사진 | `evangelism-team-2.jpg` | 찬양 밴드 연습 |
| 3번째 사진 | `evangelism-team-3.jpg` | 禱告 기도하다 슬라이드 |
| 4번째 사진 | `evangelism-team-4.jpg` | 줌 미팅 - 팀 교제 |
| 5번째 사진 | `evangelism-team.jpg` | **메인 이미지** - 야외 크리스마스 행사 전도 장면 |
| 6번째 사진 | `evangelism-team-6.jpg` | 줌 미팅 - 팀원들 |

### 2️⃣ Supabase 데이터베이스 업데이트

1. **Supabase 대시보드 접속**
   - https://supabase.com/dashboard 에 로그인
   - Gospel Meeting 프로젝트 선택

2. **SQL Editor 열기**
   - 왼쪽 메뉴에서 "SQL Editor" 클릭

3. **SQL 실행**
   - 다음 파일의 내용을 복사하여 실행:
   ```
   lib/database/migrations/add_evangelism_team_task.sql
   ```

### 3️⃣ 확인

1. 웹사이트 접속: http://localhost:3000/tasks
2. **전도팀** 카드가 표시되는지 확인
3. 전도팀 카드 클릭하여 상세 페이지로 이동
4. 메인 이미지와 사진 갤러리 6장이 정상적으로 표시되는지 확인

## 🎨 전도팀 정보

- **한글명:** 전도팀
- **중국어명:** 传道队
- **카테고리:** program (프로그램)
- **우선순위:** high (높음)
- **상태:** in_progress (진행 중)
- **태그:** 전도, 복음, 아웃리치, 전도훈련

## 📸 이미지 설명

### 메인 이미지 (5번째 사진)
- 야외 크리스마스 행사장에서 전도하는 모습
- 산타 모자를 쓴 팀원들이 방문객들에게 복음을 전하고 있습니다
- 축제 텐트와 부스가 보이는 현장감 있는 전도 장면

### 갤러리 이미지
1. 온라인 전도팀 회의 및 기도 모임
2. 찬양팀 연습 장면
3. 기도 슬라이드 (禱告 기도하다)
4. 전도팀 줌 미팅 및 교제
5. 야외 전도 활동 (메인 이미지와 동일)
6. 전도팀 온라인 모임

## 💡 참고사항

- 메인 이미지: 전도팀 카드에 표시되는 대표 사진
- 갤러리 이미지: 전도팀 상세 페이지에서 슬라이드로 표시되는 사진들
- 이미지는 `.jpg` 형식으로 저장
- 파일명은 정확히 위 표와 동일하게 저장해주세요

# 중보기도팀 이미지 추가 가이드

## 📸 이미지 추가 절차

### 1단계: 이미지 파일 확인

✅ 이미지가 이미 저장되어 있습니다!
- 파일 위치: `public/images/prayer-team-1.jpg.jpg`
- 파일명에 `.jpg.jpg`가 붙어있지만 정상 작동합니다

### 2단계: Supabase에서 SQL 실행

**Supabase SQL Editor**에서 아래 SQL을 실행하세요:

```sql
-- 중보기도팀에 메인 이미지 추가

UPDATE tasks
SET 
  image_url = '/images/prayer-team-1.jpg.jpg'
WHERE title = '중보기도팀' OR title = '중보기도';

-- 결과 확인
SELECT 
  id, 
  title, 
  title_zh,
  image_url,
  images,
  array_length(images, 1) as image_count
FROM tasks 
WHERE title = '중보기도팀' OR title = '중보기도';
```

### 3단계: 웹사이트에서 확인

1. 웹사이트 새로고침 (Ctrl+F5)
2. **/tasks** 페이지에서 중보기도팀 카드에 이미지가 표시되는지 확인
3. 중보기도팀 카드를 클릭하면 상세 페이지로 이동합니다

## 📝 갤러리에 여러 사진 추가하기

상세 페이지 갤러리에 여러 장의 사진을 추가하려면:

1. **파일 저장**:
   - `prayer-team-1.jpg.jpg` (이미 있음)
   - `prayer-team-2.jpg.jpg`
   - `prayer-team-3.jpg.jpg`
   - ...

2. **SQL 업데이트** (메인 이미지 + 갤러리):

```sql
UPDATE tasks
SET 
  image_url = '/images/prayer-team-1.jpg.jpg',
  images = ARRAY[
    '/images/prayer-team-1.jpg.jpg',
    '/images/prayer-team-2.jpg.jpg',
    '/images/prayer-team-3.jpg.jpg'
  ]
WHERE title = '중보기도팀' OR title = '중보기도';
```

**설명**:
- `image_url`: /tasks 페이지 카드에 표시되는 메인 이미지
- `images`: 상세 페이지 갤러리에 표시되는 이미지 배열

## ✅ 완료!

이제 /tasks 페이지의 중보기도팀 카드에 사진이 표시됩니다! 🙏

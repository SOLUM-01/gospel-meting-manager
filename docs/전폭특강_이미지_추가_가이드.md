# 전폭특강 포스터 이미지 추가 가이드

## 📸 작업 내용

2025 윈린 성탄절 크리스마스 축제 "환영합니다" 포스터 3장을 전폭특강 사진 갤러리에 추가합니다.

## 📋 이미지 저장하기

### 1️⃣ 이미지 파일 저장

첨부하신 3장의 포스터를 다음과 같이 저장해주세요:

**저장 위치:** `public/images/` 폴더

| 첨부 이미지 | 저장할 파일명 | 설명 |
|----------|--------------|------|
| 1번째 포스터 (가로) | `gospel-training-poster-1.jpg` | 환영합니다 포스터 - 상세 일정 포함 |
| 2번째 포스터 (뒷면) | `gospel-training-poster-2.jpg` | 환영합니다 포스터 - 뒷면 |
| 3번째 포스터 (세로) | `gospel-training-poster-3.jpg` | 환영합니다 포스터 - 세로형 |

### 2️⃣ Supabase 데이터베이스 업데이트

1. **Supabase 대시보드 접속**
   - https://supabase.com/dashboard 에 로그인
   - Gospel Meeting 프로젝트 선택

2. **SQL Editor 열기**
   - 왼쪽 메뉴에서 "SQL Editor" 클릭

3. **SQL 실행**
   - 다음 파일의 내용을 복사하여 실행:
   ```
   lib/database/migrations/add_gospel_training_images.sql
   ```

### 3️⃣ 확인

1. 웹사이트 접속: http://localhost:3000/tasks
2. **전폭특강** 카드 클릭하여 상세 페이지로 이동
3. 사진 갤러리에 3장의 포스터가 정상적으로 표시되는지 확인

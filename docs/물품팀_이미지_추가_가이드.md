`public/images/` 폴더에 다음 이미지 파일들을 저장하세요.

### 메인 이미지 (팀 대표 사진)

### 갤러리 이미지 (활동 사진들)
1. **supplies-team-1.jpg** → 첨부한 사진 중 **1번째 사진** (테이블에서 여러 사람들이 물품 작업하는 장면)

## 📁 최종 파일 구조

```
public/
  └── images/
      ├── supplies-team.jpg      (메인 이미지)
      ├── supplies-team-1.jpg    (갤러리 1)
      ├── supplies-team-2.jpg    (갤러리 2)
      ├── supplies-team-3.jpg    (갤러리 3)
      ├── supplies-team-4.jpg    (갤러리 4)
      ├── supplies-team-5.jpg    (갤러리 5)
      └── supplies-team-6.jpg    (갤러리 6)
```

## 🚀 데이터베이스 업데이트

이미지를 저장한 후, Supabase SQL Editor에서 다음 파일을 실행하세요:

```
lib/database/migrations/add_supplies_team_task.sql
```

## ✅ 확인 방법

1. 웹사이트에서 `/tasks` 페이지로 이동
2. **물품팀** 카드를 클릭
3. 메인 이미지와 갤러리 사진 6장이 정상적으로 표시되는지 확인
